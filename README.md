# getsome

Get something interesting to consume on internet. A cli tool that works like StumbleUpon.

run the command:

`getsome` or `gs`

and get an interesting link to browse.

## Ideas

### Server Requirements

* A repository to keep all the links

### Other Functionalities

* Add a link:
    * `gs add LINK [tags]`
* Remove a link:
    * `gs rm LINK`
* upvote a link:
    * `gs upvote LINK`
* downvote a link:
    * `gs downvote LINK`
* add a favourite
    * `gs likes add/rm TAG`
* rm a favorites
    * `gs dislikes add/rm TAG`
